My homebrew tap repo. If you prefer nix you can directly use in-repo flakes to install the software.

### Install

```bash
brew tap sacrilegewastaken/tap
```

### Casks

- **`olovebar`** – macos customizable menu bar. aerospace required.

Install:

```bash
brew install --cask olovebar
```

### Update

```bash
brew update
brew upgrade --cask olovebar
```
